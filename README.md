# 중소기업을 위한 자동화 공정 협동로봇 

## 프로젝트 목적

재화나 인력이 충분한 대기업의 경우에는 자신의 환경에 맞는 공정 자동화 시스템을 만들 여력이 충분합니다.
하지만 중소기업의 경우 공장을 일시적으로 빌리고 설비를 설치하는 식의 문제를 해결하는 경우가 많습니다. 
저희는 이러한 중소기업에서 사용하기 적합한 유연한 시스템의 공정 자동화 로봇을 기획해서 만들게 되었습니다.
로봇이 실제 환경에서 물체를 다루기 위해서는 다음과 같은 과정이 필요합니다.

- 카메라를 통해 작업 공간을 인식
- 물체의 위치와 상태를 판단
- 로봇이 수행할 작업 순서를 결정
- 로봇팔을 제어하여 실제 조작 수행

이를 통해서 비전으로 인간이 만든 결과물을 인식해서 자동화로 만들어주는 것이 목적인 프로젝트를 진행합니다.

## 시스템 구성

전체 시스템은 크게 두 단계로 구성됩니다.

### 1. 색상 기반 큐브 정리 단계

첫 번째 단계에서는 상단 카메라를 사용하여 작업 공간을 관찰합니다.

YOLO 기반 객체 인식을 통해 빨강, 초록, 파랑 큐브를 감지하고, 각 큐브의 위치를 로봇 기준 좌표로 변환합니다.  
이후 로봇팔은 감지된 큐브를 색상별로 지정된 위치에 정리합니다.

이 단계는 무질서하게 놓인 물체를 인식하고, 정해진 기준에 따라 재배치하는 기본적인 로봇 자동화 작업을 나타냅니다.

### 2. 구조 인식 및 재현 단계

두 번째 단계에서는 측면 카메라를 사용하여 큐브들의 상대적인 배치 구조를 관찰합니다.

시스템은 큐브 사이의 위, 아래, 오른쪽 관계를 추정하고, 이를 바탕으로 목표 구조를 계산합니다.  
이후 로봇팔은 색상별로 정리된 큐브를 다시 집어 목표 구조와 유사한 형태로 적층합니다.

이 단계는 단순한 물체 정리를 넘어, 시각적으로 관찰한 구조를 해석하고 로봇 동작으로 재현하는 것을 목표로 합니다.

## 전체 동작 흐름

```text
카메라 입력
    ↓
YOLO 기반 객체 인식
    ↓
물체 위치 및 구조 관계 추정
    ↓
로봇 작업 계획 생성
    ↓
시리얼 통신을 통한 로봇 제어
    ↓
픽앤플레이스 동작 수행
    ↓
큐브 정리 및 구조 재현
```
